النظرية العلمية

المحتوى الحراري (ΔH)

هو كمية الطاقة الحرارية المنتقلة في التفاعل الكيميائي عند ضغط ثابت. يقاس بوحدة الكيلوجول/مول (kJ/mol).

التفاعل الطارد للحرارة

تفاعل يطلق طاقة إلى المحيط (ΔH سالبة). ترتفع درجة حرارة الوسط المحيط مثل احتراق الوقود.

التفاعل الماص للحرارة

تفاعل يمتص طاقة من المحيط (ΔH موجبة). تنخفض درجة حرارة الوسط المحيط مثل تفاعلات التحلل.

أنواع التفاعلات

تفاعلات طاردة للحرارة

ΔH < 0

تفاعلات ماصة للحرارة

ΔH > 0

قانون حساب ΔH

ΔH = Hمنتجات - Hمتفاعلات

حيث H هي المحتوى الحراري للمواد

محاكاة التفاعلات الماصة والطاردة للحرارة

تفاعل هيدروكسيد الصوديوم مع حمض الهيدروكلوريك (تفاعل طارد للحرارة) وتفكك كربونات الكالسيوم (تفاعل ماص للحرارة)

التحكم في التجربة

ملاحظة: اضغط على "بدء التفاعل" لرؤية تأثير تغيير المعلمات على المحتوى الحراري.

حساب المحتوى الحراري

ΔH = H(NaCl + H₂O) - H(NaOH + HCl)

نتيجة الحساب:

-57.1 kJ/mol

القيمة السالبة تشير إلى تفاعل طارد للحرارة (ينطلق منه طاقة)

تغير درجة الحرارة المقاس:

+0.0°C

منحنى الطاقة

دليل تعليمي

تفاعلات طاردة للحرارة

احتراق الوقود: CH₄ + 2O₂ → CO₂ + 2H₂O (ΔH = -890 kJ/mol)

التعادل الحمضي القاعدي: NaOH + HCl → NaCl + H₂O (ΔH = -57.1 kJ/mol)

تفاعلات التكوين: تكوين الماء من عناصره

تفاعلات ماصة للحرارة

تفكك كربونات الكالسيوم: CaCO₃ → CaO + CO₂ (ΔH = +178 kJ/mol)

التمثيل الضوئي: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂ (ΔH = +2800 kJ/mol)

تبخر الماء: H₂O(l) → H₂O(g) (ΔH = +44 kJ/mol)

خطوات حساب المحتوى الحراري:

  1. تحديد المحتوى الحراري للتكوين القياسي (ΔH°f) لكل مادة
  2. حساب مجموع ΔH°f للمنتجات
  3. حساب مجموع ΔH°f للمتفاعلات
  4. تطبيق القانون: ΔH° = ΣΔH°f(المنتجات) - ΣΔH°f(المتفاعلات)
  5. تحديد إشارة النتيجة: سالبة = طارد للحرارة، موجبة = ماص للحرارة

مثال عملي: احتراق الميثان

CH₄ + 2O₂ → CO₂ + 2H₂O

ΔH° = [ΔH°f(CO₂) + 2×ΔH°f(H₂O)] - [ΔH°f(CH₄) + 2×ΔH°f(O₂)]

ΔH° = [(-393.5) + 2×(-285.8)] - [(-74.8) + 2×(0)] = -890.3 kJ/mol
